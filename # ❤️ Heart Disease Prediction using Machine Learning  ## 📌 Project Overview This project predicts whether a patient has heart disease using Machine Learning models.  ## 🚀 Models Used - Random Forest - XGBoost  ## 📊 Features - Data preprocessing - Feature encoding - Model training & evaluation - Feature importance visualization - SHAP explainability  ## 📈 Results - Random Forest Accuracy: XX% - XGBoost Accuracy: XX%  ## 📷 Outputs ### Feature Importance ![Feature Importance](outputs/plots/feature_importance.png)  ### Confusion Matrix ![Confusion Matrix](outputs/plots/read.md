# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Overview
This project builds a machine learning pipeline to predict the presence of heart disease based on clinical and demographic features. It demonstrates an end-to-end workflow including data preprocessing, model training, evaluation, and explainability.

---

## 🎯 Objectives
- Analyze and preprocess medical data
- Train and compare machine learning models
- Evaluate performance using standard metrics
- Interpret model predictions using SHAP

---

## 🧠 Models Implemented
- **Random Forest Classifier** — robust ensemble model for baseline performance  
- **XGBoost Classifier** — gradient boosting model for improved accuracy  

---

## ⚙️ Project Pipeline
1. Data Cleaning & Missing Value Handling  
2. Categorical Feature Encoding  
3. Train-Test Split (80:20)  
4. Feature Scaling using StandardScaler  
5. Model Training (Random Forest & XGBoost)  
6. Model Evaluation (Accuracy, Precision, Recall, F1-score)  
7. Feature Importance Analysis  
8. Model Explainability using SHAP  

---

## 📊 Results

| Model           | Accuracy |
|----------------|----------|
| Random Forest  | **89%**  |
| XGBoost        | **85%**  |
